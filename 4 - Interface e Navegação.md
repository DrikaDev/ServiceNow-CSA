## 🖥️ Interface e Navegação: Experiência do Usuário (UI/UX)

A interface do ServiceNow é projetada para facilitar o acesso e a manipulação de dados através de três componentes essenciais:  

- **Formulários (Forms):** Visualização detalhada de um único registro.
  É onde ocorrem as interações diretas, como preenchimento de campos, execução de UI Policies e visualização de atividades (Activity Stream).  

- **Listas (Lists):** Exibição tabular de múltiplos registros de uma tabela.  
  Permite edições rápidas em massa e uma visão macro do fluxo de trabalho.  

- **Filtros (Condition Builder):** Ferramenta poderosa para segmentar dados com precisão.
  Essencial para a criação de Reports, Dashboards e para a localização rápida de informações críticas.  

### 👉🏻 Diferencial Técnico: 
O domínio do Condition Builder (Filtros) é o que permite a configuração de gatilhos em ferramentas de automação, como o Flow Designer, garantindo que as ações 
certas ocorram para os registros certos.
